Cradle
======

Cradle is a code generator for a C API specified by the cradle idl.

Projects
--------

* [cradle.highlight](cradle.highlight) - Syntax highlighter for cradle idl files.
* [crates/codegen-rust](crates/codegen-rust) - Code generator for Rust bindings.
* [crates/cradle-cli](crates/cradle-cli) - Command line interface for the cradle code generator.
* [crates/cradle-idl](crates/cradle-idl) - Cradle idl parser and validator.
* [crates/example](crates/example) - Example library to demonstrate cradle features.
